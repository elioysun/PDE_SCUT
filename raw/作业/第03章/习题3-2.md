# 习题 3-2

## 1.

求解弦振动方程的下列混合问题：

(2)

$$
\begin{cases}
u_{tt} - u_{xx} = 0, & 0 < x < l,\quad t > 0, \\
\left.u\right|_{t=0} = x^2 - 2lx,\quad \left.u_t\right|_{t=0} = 0, & 0 \le x \le l, \\
\left.u_x\right|_{x=0} = \left.u\right|_{x=l} = 0, & t \ge 0;
\end{cases}
$$

(4)

$$
\begin{cases}
u_{tt} - a^2u_{xx} = 0, & 0 < x < l,\quad t > 0, \\
\left.u\right|_{t=0} = x^2(x-l)^2,\quad \left.u_t\right|_{t=0} = 0, & 0 \le x \le l, \\
\left.u_x\right|_{x=0} = \left.u_x\right|_{x=l} = 0, & t \ge 0.
\end{cases}
$$

## 2.

求解热传导方程的下列混合问题：

(1)

$$
\begin{cases}
u_t - u_{xx} = 0, & 0 < x < 1,\quad t > 0, \\
\left.u\right|_{t=0} =
\begin{cases}
x, & 0 \le x \le \dfrac{1}{2}, \\
1-x, & \dfrac{1}{2} < x \le 1,
\end{cases}
& \\
\left.u\right|_{x=0} = \left.u\right|_{x=1} = 0, & t \ge 0;
\end{cases}
$$

(3)

$$
\begin{cases}
u_t - u_{xx} = 0, & 0 < x < 1,\quad t > 0, \\
\left.u\right|_{t=0} = 3 + \cos \pi x + 4\cos 2\pi x, & 0 \le x \le 1, \\
\left.u_x\right|_{x=0} = \left.u_x\right|_{x=1} = 0, & t \ge 0.
\end{cases}
$$

## 4.

求解下列混合问题：

(1)

$$
\begin{cases}
u_{tt} - (u_{xx} + u_{yy}) = 0, & 0 < x,y < \pi,\quad t > 0, \\
\left.u\right|_{t=0} = 3\sin x \cdot \sin 2y,\quad
\left.u_t\right|_{t=0} = 5\sin 3x \cdot \sin 4y, & 0 \le x,y \le \pi, \\
\left.u\right|_{x=0,\pi} = \left.u\right|_{y=0,\pi} = 0, & t \ge 0;
\end{cases}
$$

## 9.

求解定解问题：

$$
\begin{cases}
\Delta_2 u = 0, & 1 < r < 2,\quad 0 < \theta < \pi, \\
\left.u\right|_{r=1} = \sin\theta,\quad \left.\dot{u}\right|_{r=2} = 0, \\
\left.u\right|_{\theta=0} = \left.u\right|_{\theta=\pi} = 0.
\end{cases}
$$

## 10.

求解边值问题：

$$
\begin{cases}
u_{xx} + u_{yy} = 0, & 0 < x < 1,\quad 0 < y < \infty, \\
\left.u\right|_{x=0} = \left.u\right|_{x=1} = 0,\quad \left.u\right|_{y=0} = x(1-x), \\
\lim\limits_{y\to\infty} u = 0.
\end{cases}
$$
